Velion Commands

Explore the powerful commands offered by Velion to enhance your Discord server. Whether you're looking for utility, fun, moderation, or premium features, Velion has you covered.

/ping

Shows the latency of the bot and server.

/userinfo

Shows information about a user.

/serverinfo

Displays information about the server.

/avatar

Displays a user's avatar.

/define

Define a word.

/translate

Translate text to another language.

/generate_qr

Generate a QR code for a given link.

/weather

Get weather information.

/whois

Retrieves detailed information about a user.

/channelinfo

Shows details about a specific channel.

/poll

Creates a poll with up to 10 options.

/presult

Displays the results of a poll.

/dice_roll

Rolls a dice for you.

/coinflip

Flips a coin for you.

/cat

Sends a picture or GIF of a cat.

/dog

Sends a picture or GIF of a dog.

/trivia

Starts a trivia quiz game.

/joke

Tells a random joke.

/8ball

Ask the magic 8-ball a question.

/meme

Sends a random meme from the internet.

/rps

Play rock-paper-scissors against the bot.

/gif

Search for a random GIF.

/ban

Bans a user from the server.

/kick

Kicks a user from the server.

/mute

Mutes a user.

/unmute

Unmutes a user.

/warn

Warns a user.

/lock

Locks a channel.

/unlock

Unlocks a channel.

/slowmode

Enables slow mode in a channel.

/custom_embed

Create a custom embed message.

/giveaway

Start a giveaway in the server.

/poll

Start a poll with custom styling.

/autorespond

Create an autoresponse for specific keywords.

/filter

Filter out inappropriate language.
